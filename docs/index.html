Verify your download

Every artifact is signed with minisign. Download the matching .minisig file and verify with the DozenOS public key: 

minisign -Vm dozenos-<version>-<flavor>-amd64.iso \
  -P RWRPPSRhVv7PW8vNKG1RM8ME1Tf7zQvtVBbfVjI3jcLmqJftyLjal7tq

The key is also published as minisign.pub in the release repository, and machine-readable build metadata for the latest release is available as version.json.

Nightly builds are rolling development snapshots, rebuilt automatically whenever any DozenOS source mirror changes (plus a weekly heartbeat to pick up Debian security updates). They receive automated ISO smoke testing but no manual QA — use them accordingly.

This page reads the GitHub Releases of dozenos/dozenos-nightly-build live, so it always shows the current build with no manual updates.

DozenOS is free software: all source code is available in the dozenos GitHub organization under the GNU GPL.
